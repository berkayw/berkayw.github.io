About

I developed this project to experiment with physics-driven movement in FPS games.

Trailer

Description

This is a physics-based FPS movement system inspired by games like Counter-Strike (bhopping, surfing) and Titanfall (wall running, sliding). It utilizes a Rigidbody-based controller to achieve smooth and responsive movement mechanics such as slope movement, sprinting, crouching, sliding on slopes, wall running and wall jumping. Overall, this project delivers a smooth and momentum-driven movement experience.
